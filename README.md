## Architecture

This adapter creates a bridge between your local Tiger-Monkey installation and iCloud Drive:

1. **Monitors iCloud Drive** for new case folders
2. **Syncs files** to local TM directory structure
3. **Integrates seamlessly** with Dashboard workflow
4. **Runs as macOS service** with automatic startup
