Attorney Notes Template

Copy this template structure to create consistent attorney notes for case processing.

This template provides the required field labels for consistent case data extraction. Copy and paste into your document editor, then fill in the actual case information.

Usage Instructions:

  • Copy this template structure to your preferred document editor
  • Fill in the actual case information after each label
  • Save the completed file as "Atty_Notes.txt" in your case folder
  • The Satori AI system will automatically extract the labeled information
